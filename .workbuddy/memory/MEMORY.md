## 部署说明

### 推荐方案：Netlify 拖拽上传（最简单）
1. 访问 https://www.netlify.com
2. 注册账户
3. 点击 "Add new site" → "Deploy manually"
4. 拖拽以下文件：index.html、map.html、*.json、*.toml
5. 5秒内获得免费域名（如：paris-museum.netlify.app）

### 专业方案：GitHub + Netlify 自动部署
1. GitHub 上创建仓库 paris-museum-pass
2. 双击 deploy.bat（Windows）或运行 deploy.sh（Mac/Linux）
3. 在 Netlify 连接 GitHub 仓库
4. 后续自动部署（git push 触发）

### 最快方案：GitHub Pages
1. 创建仓库 {username}.github.io
2. 上传文件
3. 访问 https://{username}.github.io 立即生效
