🚀 快速部署（3 种方式）
─────────────────────────────────────────────────────────────

【方式1】最简单：Netlify 拖拽上传（5 分钟）
────────────────────────────────────────
1. 访问 https://www.netlify.com 注册账户
2. 点击 "Add new site" → "Deploy manually"
3. 把以下文件拖上去：
   - index.html
   - map.html
   - 其他 .json、.toml 文件
4. 等待 5 秒 → 网站上线！🎉
   会得到一个免费域名，如：paris-museum.netlify.app

【方式2】最专业：GitHub + Netlify 自动部署（7 分钟）
─────────────────────────────────────────────────────
1. 在 GitHub 上创建新仓库：paris-museum-pass
   https://github.com/new
   
2. Windows 用户：双击 deploy.bat 脚本自动上传
   Mac/Linux 用户：bash deploy.sh
   
3. 按照脚本提示输入 GitHub 用户名、邮箱等信息
   脚本会自动上传所有文件到 GitHub

4. 在 Netlify 上连接 GitHub 仓库自动部署
   https://app.netlify.com
   
5. 以后修改文件 → git push → Netlify 自动更新！

【方式3】最快：GitHub Pages（5 分钟）
────────────────────────────
1. 创建名为 {你的用户名}.github.io 的仓库
2. 把所有文件上传
3. 访问 https://{用户名}.github.io 立即生效

❓ 常见问题
─────────────────────────────────────────────────────────────
Q: 怎么更新网站内容？
A: 用文本编辑器打开 HTML 文件 → 修改 → 保存
   如果用了 GitHub，还需 git push 才能同步

Q: 如何添加更多景点？
A: 编辑 map.html 中的 sites 数组，参照现有景点格式添加

Q: 网站可以用几年？
A: Netlify/Vercel 承诺永久免费，这类服务一般不会关闭

Q: 可以自定义域名吗？
A: 可以。购买域名后，在 Netlify 中配置 DNS 即可

Q: 国内用户能访问吗？
A: 可以，速度很快。Netlify 在亚太有 CDN 节点

Q: 可以添加支付功能吗？
A: 可以（后续需要）。可集成 Stripe/支付宝等
